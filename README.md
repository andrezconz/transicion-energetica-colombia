# Transición Energética en Colombia

Proyecto de análisis y simulación desarrollado en el marco del curso 
Gestión de Proyectos de Investigación y Ciencia Abierta.

## Estructura del Proyecto

- data/raw/: datos sin procesar
- data/processed/: datos transformados
- src/: funciones reutilizables
- scripts/: ejecución de análisis
- results/: figuras y tablas generadas
